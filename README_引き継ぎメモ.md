# Python 3 エンジニア認定基礎試験 オリジナル問題集（全200問プロジェクト）

## 1. プロジェクト概要
- 目的: 試験対策 兼 高校「情報Ⅰ/Ⅱ」の授業用演習教材（Google Colab）
- 全200問を【毎回15問ずつ】段階的に作成。フォーマットは全回共通。
- 各章ごとに「問題集（配布用）」と「解答と解説（先生用）」の2種を分けて管理。

## 2. フォルダ / ファイル構成
```
Python 3 Certified Member Basic Exam/
├── 01_問題集/
│   ├── [問題]03章_気楽な入門編_Q01-15.ipynb      （解答・解説なし。生徒配布用）
│   └── [問題]04章_制御フロー編_Q16-30.ipynb      （解答・解説なし。生徒配布用）
└── 02_解答と解説/
    ├── [解答解説]03章_気楽な入門編_Q01-15.ipynb  （解答・解説・完成コードあり。先生用）
    └── [解答解説]04章_制御フロー編_Q16-30.ipynb  （解答・解説・完成コードあり。先生用）
```

## 3. 進捗状況
- 第1弾: 第3章 気楽な入門編（数値・文字列・リスト）問1〜問15 … 完了
  - 各ファイル 31セル（タイトル1 + 問題markdown15 + コード15）
- 第2弾: 第4章 制御フロー編（if / for / while / range / break・continue・else）問16〜問30 … 完了
  - 各ファイル 31セル（タイトル1 + 問題markdown15 + コード15）
  - 出題内容: if/elif/else、for+range、range(start,stop,step)、while、break、continue、
    for-else（素数判定含む）、pass、enumerate、while True+break、ネストfor、in演算子、
    while-else、リスト内包表記
- 第3弾以降: 問31〜問200まで継続

## 4. 出力フォーマット規約（毎回厳守）
- 1問 = 「問題markdownセル」+「穴埋めコードセル」。空欄は `___`（アンダーバー3つ）。
- 解答解説版は「解答markdownセル」+「正解コードセル」。
- 解説はPythonチュートリアルの仕様に準拠。

## 5. GitHub 連携メモ（コミット/プッシュ手順）
- 生成した2ファイルを 01_問題集 / 02_解答と解説 フォルダに配置。
- コミット例:
```
git add "01_問題集/[問題]04章_制御フロー編_Q16-30.ipynb" "02_解答と解説/[解答解説]04章_制御フロー編_Q16-30.ipynb"
git commit -m "第4章 制御フロー編 問16-30（問題集/解答解説）を追加"
git push origin main
```
- 差分を見やすく保つため、コミット前のセル出力クリアを推奨。

## 6. 次回の作業再開方法（引き継ぎ）
- 「第3弾（問31〜問45）を、これまでと同じフォーマットで」と依頼すればOK。
- テーマ未指定なら第5章（データ構造: リスト詳細・タプル・集合・辞書 など）へ進む方針。
- ファイル名は [問題]05章_XXX_Q31-45.ipynb / [解答解説]05章_XXX_Q31-45.ipynb の規則で。
